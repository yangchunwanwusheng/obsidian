## CCF 推荐期刊/会议

### A 类（冲刺目标）

CCF A 类是计算机科学领域最高级别的会议和期刊，录用率通常较低但学术声誉极高。

| **CVPR** | 计算机视觉 | ~25% | 单轮（Rebuttal） | ⭐⭐⭐⭐⭐ | 视觉领域顶会，竞争激烈 |
| **ICCV** | 计算机视觉 | ~25% | 单轮（Rebuttal） | ⭐⭐⭐⭐⭐ | 与 CVPR 齐名，两年一届 |
| **ECCV** | 计算机视觉 | ~25% | 单轮（Rebuttal） | ⭐⭐⭐⭐⭐ | 欧洲顶会，奇数年举办 |
| **NeurIPS** | 机器学习 | ~22% | 单轮（Rebuttal） | ⭐⭐⭐⭐⭐ | ML 领域最权威会议 |

> [!warning] A 类投稿建议
> A 类会议/期刊对本科生来说挑战很大，建议有导师深度指导的情况下尝试。如果没有充足的实验支撑和创新的贡献点，不建议直接冲刺 A 类，而是从 B 类积累经验。

## 本科生特别友好的期刊

以下期刊/会议对本科生投稿特别友好，录用率较高，审稿速度较快，是积累第一篇论文经验的理想选择。

### 第一梯队：强烈推荐（录用率 30-40%，审稿 2-4 个月）

| 期刊/会议 | 方向 | 为什么适合本科生 |
|-----------|------|----------------|
| **Applied Intelligence** (C 类) | 应用 AI | 审稿快，接收率高，偏应用不要求太多理论创新 |
| **Soft Computing** (C 类) | 软计算 | 接收率友好，对实验新颖性有一定容忍度 |
| **IJCNN** (C 类会议) | 神经网络 | 会议规模大，接收率 30-35%，可以快速获得反馈 |
| **ICPR** (C 类会议) | 模式识别 | 接收率 35-40%，适合视觉方向的第一篇 |
| **ACML** (C 类会议) | 机器学习 | 亚洲举办，相对友好 |

### 第二梯队：推荐尝试（录用率 25-30%，审稿 3-6 个月）

| 期刊/会议 | 方向 | 为什么适合本科生 |
|-----------|------|----------------|
| **Neurocomputing** (B 类) | 神经计算 | B 类中审稿最快的，接收率 25-30% |
| **Expert Systems with Applications** (B 类) | 专家系统 | 偏应用，B 类中的"友好派" |
| **WACV** (B 类会议) | 视觉应用 | 应用导向，接收率相对友好 |

### 本科生投稿策略清单

- [ ] 选择偏应用的方向（而非纯理论），降低创新门槛
- [ ] 在已有开源代码基础上做改进，降低实现难度
- [ ] 实验设计要完整：主实验 + 消融实验 + 可视化分析
- [ ] 论文写作要清晰规范，避免因语言问题被拒
- [ ] 投稿前找导师或学长帮忙审读
- [ ] 如果被拒，认真阅读审稿意见并修改后投下一个目标

## 投稿策略建议

### 阶梯式投稿策略

采用"冲刺 → 主力 → 保底"三级投稿策略，最大化论文发表概率。

```
┌──────────────────────────────────────────────────┐
│  冲刺层（1-2 个目标）                              │
│  CCF B 类会议/期刊                                │
│  如：Pattern Recognition, Neurocomputing          │
│  成功率预估：20-30%                               │
│  投稿时机：论文质量较高，有明确创新点              │
├──────────────────────────────────────────────────┤
│  主力层（2-3 个目标）                              │
│  CCF B/C 类期刊                                   │
│  如：ESWA, KBS, Applied Intelligence              │
│  成功率预估：30-50%                               │
│  投稿时机：冲刺被拒后修改，或直接投稿              │
├──────────────────────────────────────────────────┤
│  保底层（1-2 个目标）                              │
│  CCF C 类会议/期刊，或非 CCF 但正规期刊           │
│  如：IJCNN, Soft Computing, NCA                   │
│  成功率预估：40-60%                               │
│  投稿时机：前两轮被拒后，快速发表积累经验          │
└──────────────────────────────────────────────────┘
```

### 按论文质量的投稿决策树

```
论文完成
  │
  ├─ 创新性强 + 实验充分 + 导师认可
  │   → 冲刺 B 类（Pattern Recognition / KBS）
  │
  ├─ 有一定创新 + 实验完整
  │   → 主力 B/C 类（Neurocomputing / ESWA / Applied Intelligence）
  │
  ├─ 改进为主 + 实验完整
  │   → 保底 C 类（Soft Computing / NCA）
  │
  └─ 工作不够完整
      → 继续完善实验，不要急于投稿
```

### 时间规划建议

| 目标会议/期刊 | 建议开始准备时间 | 投稿时间 | 结果时间 |
|-------------|----------------|----------|----------|
| NeurIPS 2026 | 2026年2月 | 2026年5月 | 2026年9月 |
| EMNLP 2026 | 2026年3月 | 2026年6月 | 2026年10月 |
| AAAI 2027 | 2026年4月 | 2026年7-8月 | 2026年11月 |

## OA vs 非OA 对比表

| 对比维度 | OA（开源获取） | 非OA（订阅制） |
|----------|---------------|---------------|
| **版面费** | $1500-3500（约 1-2.5 万人民币） | 通常免费，个别收少量彩图费 |
| **作者保留版权** | 通常保留（CC BY/CC BY-NC） | 通常签署版权转让 |
| **录用难度** | 部分期刊 OA 版录用率略高（但好的 OA 期刊仍然严格） | 同等水平 |
| **审稿速度** | OA 期刊通常更快（2-4 个月） | 传统期刊较慢（3-8 个月） |
| **学术声誉** | 顶级 OA（如 Nature Communications）声誉极高；一般 OA 需甄别 | 传统顶刊声誉稳定 |
| **可见度/引用** | 高——任何人都能免费阅读 | 低——需要机构订阅 |
| **适合场景** | 有经费支持、希望快速传播 | 无额外经费、追求传统声誉 |

### OA 期刊推荐（适合有限预算）

| 期刊 | 版面费 | 方向 | 备注 |
|------|--------|------|------|
| **IEEE Access** | $1,750 | 综合 | OA 中声誉较好，审稿快 |
| **PLOS ONE** | $1,929 | 综合 | 接收率较高，偏重方法正确性 |
| **Sensors (MDPI)** | $2,600 | 传感器/检测 | MDPI 出版，审稿快但声誉有争议 |
| **Electronics (MDPI)** | $2,400 | 电子/AI | MDPI 出版，接收率较高 |
| **Machine Learning and Knowledge Extraction (MDPI)** | $1,400 | ML | 价格相对友好 |
| **Applied Sciences (MDPI)** | $2,200 | 应用科学 | 范围广，接收率较高 |

> [!caution] MDPI 注意事项
> MDPI 旗下期刊审稿速度快、接收率高，但在部分学术圈存在争议（被批评为"水刊"）。建议：
> - 第一篇论文可以考虑 MDPI 积累经验
> - 后续论文应尽量投传统出版社（Elsevier, Springer, IEEE）
> - 投稿前咨询导师意见

### 免费/低价发表方案

| 方案 | 途径 | 备注 |
|------|------|------|

## 投稿前检查清单

每次投稿前，使用以下清单确保质量：

### 论文内容
- [ ] 核心贡献点是否明确（2-3 个）
- [ ] 创新点是否有足够的实验支撑
- [ ] Related Work 是否完整覆盖了关键文献
- [ ] Method 部分是否有足够的细节供复现
- [ ] Experiment 部分是否包含主实验 + 消融实验

### 格式规范
- [ ] 是否使用目标会议/期刊的 LaTeX 模板
- [ ] 页数是否在限制范围内（通常 8-10 页正文 + 参考文献）
- [ ] 图表是否清晰（300dpi 以上，矢量格式优先）
- [ ] 参考文献格式是否正确
- [ ] 是否有匿名处理（会议投稿通常需要盲审）

### 语言质量
- [ ] 拼写和语法检查
- [ ] 专业术语使用是否一致
- [ ] 缩写首次出现是否展开
- [ ] 避免口语化表达

### 投稿材料
- [ ] 论文 PDF
- [ ] 补充材料（代码、额外实验、视频等）
- [ ] Cover Letter / 投稿信
- [ ] 作者信息和 ORCID
- [ ] 利益冲突声明
